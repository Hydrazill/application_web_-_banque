Etape pour demarre l'application:
    
    1- Installer flask si ce n'est pas encore fait en executant la commande:
        " pip install flask " sur windows
        " pip3 install flask " sur ubuntu
        (nb: vous devez au prealable avoir installer python sur votre machine)
    
    2- Demarre le serveur local en executant la commande:
        " python main.py " sur windows
        " python3 main.py " sur ubuntu

    3- Demarrer le navigateur et dans la barre de recherche saisisser l'URL:
        " http://127.0.0.1:5000 " ou " http://localhost:5000 " ou " localhost:5000 "
